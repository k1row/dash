Features

DASH is a simple and responsive Sass/Compass-based framework
for faster and easier web development.

  • Simple and Clean

    DASH is a simple and clean framework. You can customize and development easier and faster.

  • Sass/Compass

    DASH builds on Sass/Compass. Some variable and mixin is available from the beginning.

  • Responsive

    DASH is cross device compatible. You can create a website viewable from all devices.

  • Icon Font

    Icon font, Entypo is included in DASH, so you don't need prepare icon.

Usage

You can create a web site with simple code. Grid system, some variable, mixin,
component are included. More usage is written in Read Me.
